# 0x03. Python - Everything is object

This project explores how Python treats variables as references to objects,
covering mutability, aliasing, identity vs equality, and how the interpreter
manages memory under the hood.

## Files

- `0-answer.txt` to `28-answer.txt`: short-answer files exploring object
  identity, mutability, and aliasing behavior in Python.
- `19-copy_list.py`: a function that returns a shallow copy of a list.
